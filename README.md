 I made this engine for fun\

## How to build:
 ### in linux:
 example: build example, yes\

dependencies:\
`sfml open-al`
build:\
```
 cd ./examples/cube/  
 mkdir build && cd build  
 cmake .. && make  
```
  ### in windows
```
cd .\examples\cube\
mkdir build && cd build
cmake .. && make
```
